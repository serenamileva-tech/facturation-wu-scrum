# facturation-wu-scrum
Système de facturation de Western Union avec la méthode Scrum

## 📌 Description
Application web inspirée de Western Union permettant de :
- créer des transactions
- calculer automatiquement les frais (5%)
- générer une facture
- consulter un historique

## ⚙️ Technologies
- HTML / CSS / JavaScript
- Méthode Agile Scrum
- Trello pour gestion

## 👥 Équipe
- MABETE
- MAKOUKA
- Itarcy Peniel

## 🚀 Fonctionnalités
- Création de transaction
- Calcul automatique
- Génération facture
- Historique
- Recherche

## 📊 Méthodologie
Projet réalisé avec Scrum en 4 sprints :
1. Analyse
2. Développement UI
3. Logique métier
4. Finalisation
